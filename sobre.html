Sobre Mim

Meu nome é Alan Queirós de Goes e trabalho com tecnologia há mais de 20 anos. Ao longo dessa trajetória, tive a oportunidade de adquirir uma vasta experiência em diversas áreas do setor, o que me permitiu construir uma carreira sólida e desafiadora. Atualmente, atuo como técnico de nível 2 no Senado Federal, oferecendo suporte técnico e soluções eficazes para os sistemas e infraestrutura de TI.

Minha jornada no mundo da tecnologia começou ainda cedo, com um interesse natural por computadores e seus componentes. Com o tempo, essa paixão se transformou em uma carreira focada na solução de problemas tecnológicos, na manutenção de sistemas e na inovação de processos. Ao longo dos anos, busquei sempre aprimorar meus conhecimentos para lidar com as mudanças e as novas exigências do mercado de TI.

Hoje, minha principal meta é me tornar um especialista em desenvolvimento Full Stack. Estou focado em aprender e dominar tecnologias essenciais como Python, JavaScript, HTML, CSS, PHP e WordPress. Acredito que essas ferramentas são fundamentais para criar soluções web completas, do front-end ao back-end, e estou constantemente me aprimorando para integrar esses conhecimentos de maneira eficaz e criativa.

Além da minha experiência profissional, estou sempre em busca de novos desafios que me permitam expandir meus horizontes. Fazer cursos, trabalhar em projetos pessoais e colaborar com colegas da área são formas de me manter atualizado. Meu objetivo é continuar evoluindo no campo da tecnologia e, no futuro, contribuir para a criação de soluções inovadoras que possam beneficiar tanto o ambiente corporativo quanto a sociedade como um todo.

Minhas Linguagens de Programação

Python

Python é uma linguagem poderosa e versátil, ideal para automação, análise de dados, desenvolvimento web e muito mais. Estou me aprofundando para utilizá-la de forma eficiente em projetos práticos.

JavaScript

JavaScript é essencial para o desenvolvimento de páginas dinâmicas e interativas. Estou explorando seus frameworks modernos para criar aplicações ricas e funcionais.

HTML

HTML é a base da web, permitindo estruturar o conteúdo de páginas. Estou dominando as melhores práticas para criar sites semânticos e acessíveis.

CSS

CSS é fundamental para estilizar e tornar os sites visualmente atraentes. Estou me especializando em técnicas avançadas para criar layouts responsivos e modernos.

PHP

PHP é uma linguagem poderosa para o desenvolvimento de sites dinâmicos. Estou estudando suas capacidades em backend para criar aplicações completas e funcionais.

WordPress

WordPress é uma das plataformas mais populares para criação de sites e blogs. Estou aprendendo a customizar temas e desenvolver plugins para oferecer soluções completas para meus clientes.
